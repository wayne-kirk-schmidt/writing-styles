# t_resume.md  
Resume Template — Table Version

| Section | Name | Description |
|---------|------|-------------|
| **1** | Contact Information | Provides essential details so employers can reach you quickly. |
| 1.1 | Name | Full legal or professional name. |
| 1.2 | Contact Details | Email, phone number, and location (city/country only). |
| 1.3 | Links | Optional LinkedIn, website, GitHub, or portfolio links. |
| **2** | Professional Summary | A concise 2–3 sentence overview of your background and strengths. |
| 2.1 | Core Identity | Your professional label (e.g., “AI Architect”). |
| 2.2 | Strengths | 2–3 high-value strengths or focus areas. |
| 2.3 | Direction | The impact you aim to deliver in your next role. |
| **3** | Skills | Key technical, functional, and soft skills. |
| 3.1 | Technical Skills | Tools, technologies, programming languages, cloud platforms. |
| 3.2 | Functional Skills | Leadership, analysis, architecture, operations, etc. |
| 3.3 | Soft Skills | Communication, teamwork, problem-solving. |
| **4** | Work Experience | Chronological list of roles and achievements. |
| 4.1 | Job Title & Company | The role and the organization. |
| 4.2 | Dates of Employment | Start and end dates (month/year). |
| 4.3 | Key Responsibilities | Main activities and role expectations. |
| 4.4 | Achievements | Quantified accomplishments and outcomes. |
| **5** | Education | Academic background. |
| 5.1 | Degree | Type of degree (B.A., M.S., etc.). |
| 5.2 | Institution | Name of the school or university. |
| 5.3 | Dates | Years attended or graduated. |
| **6** | Certifications | Relevant professional credentials. |
| 6.1 | Certification Name | Title of certification. |
| 6.2 | Issuing Organization | Who awarded it. |
| 6.3 | Date Earned | Month/year or year. |
| **7** | Projects | Key professional, academic, or personal projects. |
| 7.1 | Project Name | Title of project. |
| 7.2 | Description | What the project accomplished or solved. |
| 7.3 | Results | Measurable outcomes or insights. |
| **8** | Awards | Recognition for excellence. |
| 8.1 | Award Title | Name of the award. |
| 8.2 | Context | Reason for receiving it. |
| 8.3 | Date | When awarded. |
| **9** | Publications & Speaking | Optional academic or public contributions. |
| 9.1 | Publications | Titles, venues, or summaries. |
| 9.2 | Presentations | Talks, panels, conference contributions. |
| **10** | Additional Information | Optional supporting items. |
| 10.1 | Languages | Spoken or written languages with proficiency. |
| 10.2 | Interests | Optional personal interests. |
| 10.3 | Availability | Start dates or relocation notes if applicable. |

# General Report Template

This is a guideline for you to think about the writing process. 
Whtether you have more or less sections and items you want to:
* Have the writing have a structure
* Have the structure follow a flow of thought
* Have each step of the structure support the previous step

## Section Table

| Section | Name | Description |
|--------|------|-------------|
| **1** | Executive Summary | High-level overview of purpose, findings, and key recommendations. |
| 1.1 | Purpose Snapshot | Why this report exists. |
| 1.2 | Key Findings | Most important results in brief. |
| 1.3 | Key Recommendations | The main decisions or actions proposed. |
| **2** | Background and Context | Explain the situation, history, constraints, and actors involved. |
| 2.1 | Situation Overview | What led to the report. |
| 2.2 | Scope | What is included and excluded. |
| 2.3 | Stakeholders | People, teams, or groups affected. |
| **3** | Objectives | What the report intends to accomplish. |
| 3.1 | Primary Objectives | Main goals or questions. |
| 3.2 | Success Criteria | How effectiveness will be evaluated. |
| **4** | Methodology or Approach | How information was gathered and analyzed. |
| 4.1 | Data Sources | Where information came from. |
| 4.2 | Tools / Frameworks | Methods used for analysis. |
| 4.3 | Limitations | Constraints, assumptions, missing data. |
| **5** | Findings | The raw observations, results, and discovered facts. |
| 5.1 | Thematic Findings | Key themes or categories of findings. |
| 5.2 | Evidence | Supporting data or examples. |
| **6** | Analysis | Interpretation of the findings and their implications. |
| 6.1 | Interpretation | What the findings mean. |
| 6.2 | Trends / Patterns | Relationships or notable dynamics. |
| 6.3 | Risks / Opportunities | Positive or negative implications. |
| **7** | Options or Alternatives | Possible paths forward and their trade-offs. |
| 7.1 | Option A | Overview, benefits, drawbacks. |
| 7.2 | Option B | Overview, benefits, drawbacks. |
| 7.3 | Comparative Evaluation | How the options compare. |
| **8** | Recommendations | The preferred action(s) with justification. |
| 8.1 | Primary Recommendation | The clear preferred course of action. |
| 8.2 | Rationale | Why this is the optimal choice. |
| 8.3 | Mitigations | How to handle risks. |
| **9** | Implementation Considerations | Practical planning for execution. |
| 9.1 | Timeline | High-level phases or milestones. |
| 9.2 | Resources | People, tools, budget required. |
| 9.3 | Governance | Oversight roles, checkpoints, responsibilities. |
| **10** | Conclusion | Final summary and forward-looking perspective. |
| 10.1 | Summary | Reaffirm key insights. |
| 10.2 | Closing Perspective | Future implications or next steps. |
| **11** | Appendices | Supplemental information. |
| 11.1 | Data / Charts | Extended tables, figures, supporting detail. |
| 11.2 | Glossary | Definitions, terminology. |
| 11.3 | References | Citations or related documents. |

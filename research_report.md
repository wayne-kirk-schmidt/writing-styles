# Research Report Template (Table Format)

## Section Table

| Section | Name | Description |
|--------|------|-------------|
| **1** | Introduction | Define the topic, explain context, state relevance, narrow the scope. |
| 1.1 | Topic Definition | Briefly describe the subject being investigated. |
| 1.2 | Context | Explain the relevant background the reader needs. |
| 1.3 | Importance | Clarify why this topic matters now. |
| 1.4 | Focus Narrowing | Specify the exact angle your paper will address. |
| **2** | Thesis Statement | Present the core question or claim and purpose of the analysis. |
| 2.1 | Research Question | State the main question or thesis guiding the paper. |
| 2.2 | Purpose | Explain what you intend to uncover or demonstrate. |
| 2.3 | Topic Rationale | Why you chose this topic. |
| **3** | Quantitative Evidence | Present data, statistics, trends supporting your argument. |
| 3.1 | Data Presentation | Show key numbers or metrics. |
| 3.2 | Trend Interpretation | Explain what patterns or correlations appear. |
| 3.3 | Relevance | Connect the numbers directly to your thesis. |
| **4** | Qualitative Evidence | Provide interviews, observations, or case studies. |
| 4.1 | Interviews / Cases | Summaries of human perspectives. |
| 4.2 | Insights | What perspectives reveal beyond numbers. |
| 4.3 | Themes | Emergent patterns and qualitative findings. |
| **5** | Consequences if Not Addressed | Physical, economic, social, environmental impacts. |
| 5.1 | Physical Effects | Tangible or system-level impacts. |
| 5.2 | Economic Effects | Financial or business consequences. |
| 5.3 | Social Effects | Human stress, displacement, well-being impacts. |
| 5.4 | Environmental Effects | Systemic or ecological consequences. |
| **6** | Solutions | System-level and individual-level solutions. |
| 6.1 | System-Level Solutions | Actions governments or organizations can take. |
| 6.2 | Individual-Level Solutions | Actions individuals or small groups can take. |
| **7** | Community Action | How communities organize and collaborate. |
| 7.1 | Local Groups | Community networks and shared resources. |
| 7.2 | Collective Action | How community-wide efforts differ from individual action. |
| 7.3 | Resilience | How shared action strengthens preparedness. |
| **8** | Conclusion & Call to Action | Summarize findings and specify required actions. |
| 8.1 | Thesis Restatement | Reframe the original thesis with full insight. |
| 8.2 | Findings Summary | Combine quantitative + qualitative results. |
| 8.3 | Action Recommendation | Specify what should happen next and by whom. |
| 8.4 | Forward Insight | Close with a future-oriented message. |
| **9** | Reflection | Evaluate the research process and personal learning. |
| 9.1 | Quantitative Lessons | What you learned from numerical research. |
| 9.2 | Qualitative Lessons | What interviews or observations revealed. |
| 9.3 | Source Credibility | Strongest and weakest sources. |
| 9.4 | Bias Handling | How bias was located and mitigated. |
| 9.5 | Challenges | Difficult or surprising aspects. |
| 9.6 | Improvements | What to change next time. |
| 9.7 | Perspective Shift | How your understanding evolved. |
